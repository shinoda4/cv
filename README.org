** Clone

#+begin_src shell
git clone git@github.com:shinoda4/cv.git
git clone https://github.com:shinoda4/cv.git  
#+end_src

** Install Typst

#+begin_src shell
cargo install --locked typst-cli  
#+end_src

Ubuntu: =snap install typst=

macOS: =brew install typst=

Windows: =winget install --id Typst.Typst=

** Compile and watch

#+begin_src shell
typst watch resume.typ --font-path=fonts --open  
#+end_src
